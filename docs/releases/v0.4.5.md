# Tileline v0.4.5 Release Notes

Release date: 2026-03-22

## Scope

`v0.4.5` is the 2D foundation consolidation release.

This release locks a canonical project/runtime content path around `.tlpfile`,
extends scene configuration with explicit `2d|3d` separation, and finalizes
the pre-beta package flow around root-scoped project packaging.

## Highlights

- Added `.tlpfile` scene dimension separation:
  - `[project] default_dimension = 2d|3d`
  - `[scene.*] dimension = 2d|3d`
- Runtime now applies `.tlpfile` scene dimension at startup (`2d` side-view / `3d` spatial).
- Runtime project reload path now also reapplies scene mode from `.tlpfile`.
- Added root-scoped `.tlpfile` pack builder:
  - `scripts/build_tlpfile_pak.sh`
  - numbered shard output (`.part0001.pak`, `.part0002.pak`, ...)
  - default shard cap: 5 GB
- Updated pre-beta package script:
  - `scripts/package_prebeta_pak.sh` now defaults to project mode (`.tlpfile` root)
  - legacy `--src/--out` mode remains available
- Demo FBX assets normalized under `docs/demos/tlapp/` for package-root consistency.

## Included Binary

Primary binary for this release:

- `runtime` crate: `tlapp`

Release artifacts are generated into:

- `dist/v0.4.5/<target-triple>/`

Expected files:

- `tlapp` (or `tlapp.exe` on Windows)
- `SHA256SUMS`
- `build-info.txt`
- `tileline-v0.4.5-<target-triple>-tlapp.tar.gz`

## Build and Package

```bash
./scripts/release_v0.4.5.sh
```

Optional cross-target build:

```bash
TARGET_TRIPLE=aarch64-apple-darwin ./scripts/release_v0.4.5.sh
```

Project-root `.pak` build (numbered 5GB shards):

```bash
./scripts/build_tlpfile_pak.sh \
  --project docs/demos/tlapp/tlapp_project.tlpfile \
  --out-dir dist/pak \
  --base-name tlapp-assets \
  --max-shard-gb 5
```

## Run

```bash
./dist/v0.4.5/<target-triple>/tlapp \
  --project docs/demos/tlapp/tlapp_project.tlpfile \
  --scene main
```

## Notes

`v0.4.5` is the stabilization baseline before the broader `v0.5.0` independence
track (render stack cutover, MPS/ParadoxPE revision, and further optimization).
