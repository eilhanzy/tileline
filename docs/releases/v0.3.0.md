# Tileline v0.3.0 Release Notes

Release date: 2026-03-15

## Scope

`v0.3.0` is a feature release that finalizes the in-app runtime console tooling baseline and
adds module-aware engine version introspection.

Highlights:

- Added unified runtime version registry (`tileline`, `runtime`, `tl-core`, `mps`, `gms`, `mgs`, `nps`, `paradoxpe`).
- Added TLApp console version commands:
  - `version`
  - `version all`
  - `version <module>`
- Added TLApp CLI startup version flag:
  - `-V`, `--version`
- Expanded file command surface for live debugging:
  - `file.exists`, `file.head`, `file.tail`, `file.read`, `file.list`
  - `file.find`, `file.findi`, `file.findr`, `file.grep`
  - `file.tailf`, `file.watch`
- Added fail-soft bounded I/O safety rules for console file operations (workspace root guard + capped windows).

## Included Binary

Primary binary for this release:

- `runtime` crate: `tlapp`

Release artifacts are generated into:

- `dist/v0.3.0/<target-triple>/`

Expected files:

- `tlapp` (or `tlapp.exe` on Windows)
- `SHA256SUMS`
- `build-info.txt`
- `tileline-v0.3.0-<target-triple>-tlapp.tar.gz`
