# Tileline v0.1.0 Release Notes

Release date: 2026-03-14

## Scope

`v0.1.0` is the first pre-alpha release baseline for Tileline.

Highlights:

- TLApp runtime demo stabilized at high object counts (targeting 8k balls).
- ParadoxPE anti-penetration guard and post-step boundary reconciliation are integrated into the runtime loop.
- `.tlscript` / `.tlsprite` demo control path is wired as the default show path.
- Release packaging flow now exports a runnable `tlapp` binary.

## Included Binary

Primary binary for this release:

- `runtime` crate: `tlapp`

- `dist/v0.1.0/<target-triple>/`

## Build and Package

Use the release script from repository root:

```bash

```

Optional cross-toolchain preflight:

```bash
./scripts/setup_cross_toolchains.sh --target all
```

Optional cross-target build:

```bash

```
