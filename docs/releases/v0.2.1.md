# Tileline v0.2.1 Release Notes

Release date: 2026-03-15

## Scope

`v0.2.1` is a stabilization release focused on collision robustness and safer high-energy showcase behavior.

Highlights:

- Added swept/speculative broadphase candidate capture to reduce high-speed tunnel misses.
- Added speculative narrowphase contact generation for fast approach pairs.
- Added runtime hard-safety recycle path for escaped bodies when virtual barrier mode is disabled.
- Added velocity safety clamp in showcase scene tick to bound one-step tunnel risk.
- Exposed speculative collision tuning to `.tlscript`:
  - `set_speculative_sweep(enabled, max_distance)`
  - `set_speculative_contacts(enabled, contact_distance, max_prediction_distance)`
- Updated TLApp showcase scripts (desktop + mobile-safe) with safe speculative defaults.
