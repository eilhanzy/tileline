# Tileline v0.2.0

Date: 2026-03-15

Tileline `v0.2.0` continues the pre-beta transition and introduces the first built-in `.pak` packaging workflow.

## Highlights

- Added runtime `.pak` packaging core (`pack/list/unpack`) with deterministic archive order.
- Added `.pak` checksum verification and safe relative-path extraction rules.
- Added pre-beta asset packaging helper script: `scripts/package_prebeta_pak.sh`.
- Kept release output centered on a runnable `tlapp` binary.

## Quick Start

Build release artifact:

```bash
./scripts/release_v0.2.0.sh
```

Build pre-beta demo asset package:

```bash
./scripts/package_prebeta_pak.sh
```

Run TLApp:

```bash
./dist/v0.2.0/<target-triple>/tlapp --fps-cap 60 --vsync auto
```

## Notes

- This is a pre-beta track release. API/data format changes may continue through `0.2.x`.
- Runtime `.pak` mount/resolve integration is still pending; this release provides packaging toolchain and validation layer.
