# Tileline v0.3.0

Date: 2026-03-15

Tileline `v0.3.0` extends the pre-beta runtime tooling with module-aware version commands and a complete in-app file-inspection command surface.

## Highlights

- Added module-aware version registry and query path:
  - `version`
  - `version all`
  - `version <module>`
- Added startup version flag: `--version` / `-V`.
- Added runtime console file tools:
  - `file.exists`, `file.head`, `file.tail`, `file.read`, `file.list`
  - `file.find`, `file.findi`, `file.findr`, `file.grep`
  - `file.tailf`, `file.watch`
- Added bounded fail-soft file access constraints suitable for live runtime debugging.

## Quick Start

Build release artifact:

```bash
./scripts/release_v0.3.0.sh
```

Build pre-beta demo asset package:

```bash
./scripts/package_prebeta_pak.sh
```

Run TLApp:

```bash
./dist/v0.3.0/<target-triple>/tlapp --fps-cap 60 --vsync auto
```
