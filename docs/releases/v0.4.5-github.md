# Tileline v0.4.5

Date: 2026-03-22

Tileline `v0.4.5` is the 2D foundation consolidation release.
It stabilizes `.tlpfile`-driven scene composition, adds explicit `2d|3d`
scene dimension control, and finalizes project-root `.pak` packaging with
numbered shard output.

## Highlights

- `.tlpfile` now supports:
  - `default_dimension = 2d|3d` in `[project]`
  - `dimension = 2d|3d` in `[scene.*]`
- TLApp startup/reload now applies scene mode from `.tlpfile` manifest.
- Added `scripts/build_tlpfile_pak.sh`:
  - `.tlpfile` root is canonical package root
  - numbered shards (`part0001...`) with default 5GB cap
- `scripts/package_prebeta_pak.sh` now defaults to project mode while keeping legacy mode.
- Demo FBX assets moved to `docs/demos/tlapp/` to match package-root flow.

## Quick Start

Build release artifact:

```bash
./scripts/release_v0.4.5.sh
```

Build project package shards:

```bash
./scripts/build_tlpfile_pak.sh \
  --project docs/demos/tlapp/tlapp_project.tlpfile \
  --out-dir dist/pak \
  --base-name tlapp-assets \
  --max-shard-gb 5
```

Run TLApp from project manifest:

```bash
./dist/v0.4.5/<target-triple>/tlapp \
  --project docs/demos/tlapp/tlapp_project.tlpfile \
  --scene main
```
