# Tileline v0.2.1

Date: 2026-03-15

Tileline `v0.2.1` is a stability-focused pre-beta update that hardens fast-motion collision behavior.

## Highlights

- Added swept/speculative broadphase pair capture for high-speed anti-tunneling.
- Added speculative narrowphase contact generation for fast sphere/surface approach pairs.
- Added runtime hard-safety recycle path when virtual barrier is disabled.
- Added runtime collision-safety velocity clamp in bounce showcase path.
- Added `.tlscript` collision controls:
  - `set_speculative_sweep(enabled, max_distance)`
  - `set_speculative_contacts(enabled, contact_distance, max_prediction_distance)`
- Updated showcase scripts with safe speculative defaults for desktop and mobile-safe presets.

## Quick Start

Build release artifact:

```bash
./scripts/release_v0.2.1.sh
```

Build pre-beta demo asset package:

```bash
./scripts/package_prebeta_pak.sh
```

Run TLApp:

```bash
./dist/v0.2.1/<target-triple>/tlapp --fps-cap 60 --vsync auto
```
