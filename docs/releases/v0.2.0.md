# Tileline v0.2.0 Release Notes

Release date: 2026-03-15

## Scope

`v0.2.0` advances Tileline from pure pre-alpha baseline toward pre-beta readiness.

Highlights:

- Runtime `.pak` packaging pipeline is now available (`pack/list/unpack`) with deterministic archive order.
- `.pak` extraction path includes checksum verification and safe relative-path enforcement.
- Pre-beta packaging helper script is added for demo/runtime assets.
- Release packaging flow remains centered on a runnable `tlapp` binary artifact.

## Included Binary

Primary binary for this release:

- `runtime` crate: `tlapp`

Release artifacts are generated into:

- `dist/v0.2.0/<target-triple>/`

Expected files:

- `tlapp` (or `tlapp.exe` on Windows)
- `SHA256SUMS`
- `build-info.txt`
- `tileline-v0.2.0-<target-triple>-tlapp.tar.gz`

## Build and Package

Use the release script from repository root:

```bash
./scripts/release_v0.2.0.sh
```

Optional cross-toolchain preflight:

```bash
./scripts/setup_cross_toolchains.sh --target all
```

Optional cross-target build:

```bash
TARGET_TRIPLE=aarch64-apple-darwin ./scripts/release_v0.2.0.sh
```

## Pre-Beta Asset Package (`.pak`)

```bash
./scripts/package_prebeta_pak.sh
```

## Run

```bash
./dist/v0.2.0/<target-triple>/tlapp --fps-cap 60 --vsync auto
```
